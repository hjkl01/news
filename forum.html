如何看待“Linux 桌面永远没有春天”这种言论？

每年都有人预言这是 Linux 桌面的元年，但事实总是相反。除了驱动、游戏和 Adobe 等老生常谈，真正的原因更令人沮丧：未来的电脑用户可能不再只是人类，而是 AI 代理。这些代理需要依赖 Accessibility API 来操作电脑，而 macOS 凭借默认开启的无障碍支持遥遥领先。Windows 虽有强大的 UI Automation，却受制于碎片化的应用生态。反观 Linux，尽管有 AT-SPI，但权限模型混乱、窗口枚举困难…
